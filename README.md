# Ad Tag Testing Tool

A simple web-based tool to validate and preview advertising tags.

## Features

- Validate Ad Tags (Iframe / Script)
- Detect Ad Size
- Detect Tracking Pixel
- Preview Ad Creatives
- Measure Ad Load Time

## Tech Stack

HTML  
CSS  
JavaScript (Vanilla JS)
