# Ad Tag Testing Tool

A web-based tool for validating and previewing digital advertising tags before deployment.

This tool helps developers and Ad Operations teams test ad creatives, detect tracking pixels, and measure ad load performance.

---

## Features

• Validate ad tags (Iframe / Script / Image)  
• Detect ad creative size  
• Detect tracking pixels  
• Preview ads inside a secure iframe  
• Measure ad load time
